KB1 MIDI Editor - Action Bar Layout

New buttons added to both SLIDERS and SETTINGS pages for snapshot/restore and reset to defaults functionality.

SLIDERS Page - Action Bar

Enhanced MIDI editor page with snapshot restore and firmware defaults reset.

SETTINGS Page - Action Bar

Enhanced device settings page with snapshot restore, firmware defaults reset, and local changes reset.

Button Functions

  • Load from Device: Reads all settings from KB1 via BLE and captures a snapshot
  • Restore from Snapshot: Recalls last snapshot from localStorage (captured after successful load)
  • Reset to Defaults: Resets all settings to firmware defaults (requires confirmation)
  • Reset Changes: (Settings only) Reverts local changes back to device values
  • Apply to Device: Sends current settings to KB1 RAM
  • Save to Flash: Persists current RAM settings to KB1 flash memory
